CSS box model

Below are a few examples of a CSS box model. A CSS box model is a box that wraps around each HTML element that is present.

In this CSS box model, I have used multiple elements. The main elements include the background colour, width, padding, and margin. We of course also include the content in the box.

In order to have the boxes alligned side by side, I used the element float. When added to the div classes, I set the float elements to left, alligning the two boxes to the left of each other.

For each of the boxes, I have set the width to 300px, the padding to 20px, and the margin to 10px. Furthermore, I decided to make use out of box shadows, adding a drop shadow.

Each CSS box present has a different element changed about it. One has its background colour changed, and another has its corner radius changed to name a few.

Below is a diagram displaying what the CSS box model looks like:

In the diagram, we can see that it renders each element as a rectangular box. The CSS box model determines its size, position, and properties. Its basic properties are as follows:

  • Content - The content that is present inside of the box
  • Padding - The empty area between the content and the edge of the box
  • Border - A border that surrounds the padding and content
  • Margin - The empty area outside of the border

Along with these, you can change the colour of the box, its background and so on.

The content of the box contains data such as text, images, or videos. The content cannot go beyond the padding, so it stays wrapped in the center. The content portion can have a background image or a different background colour, as shown at the beginning of the page.

The paddings edge extends the content area of the box, including the elements padding. The paddings dimensions can be increased and decreased by the "padding" property. This is what wraps around it. Following this, the padding is extended by the border edge, which includes the elements border. We can change the borders thickness, with the main property of this being "border -width".

Lastly, the margins edge extends the border area. The margin is used for seperating the element from other elements that may be near or next to it. You can change the margin by the "margin" property.
