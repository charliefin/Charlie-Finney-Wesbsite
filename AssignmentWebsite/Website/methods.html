Introduction to Methods

CSS Methods

There are three main methods that are used for CSS. These are:

  • External
  • Internal
  • Inline

Each method takes a slightly different approach into displaying CSS elements. We will be looking at each method, and showing the difference between them and their roles.

Inline CSS

The inline method is used for adding unique styles to different elements. It is done by applying the "style" attribute to the chosen HTML element that you want to style. For example, within this webpage, I have given a part of the layout a grey thin border. Here is how I did it below using the "style" attribute:

<div class="col-6" style="border-left: 1px solid #ddd; border-right: 1px solid #ddd;">

The inline approach is very useful for giving seperate elements like headers and boxes unique styles, giving them a new set of rules to follow.

Internal CSS

The internal method is used for styling one single page. It allows for giving a page a unique style from the rest, instead of giving a unique style to one element. This is done by adding the "style" tag in the "head" tag of your HTML page. Here is how I have styled one page using the "style" tag:

Internal CSS is quite useful, but can become very time consuming when adding styles for every page. One great addition to internal CSS is that you can use class and ID selectors in the style sheets. For example, instead of using .class, you can instead use #methodName. This is very useful for devlopers, as its easier to know where each part of the website is stored.

External CSS

The external method is used to change the style of an entire webpage by linking an external page. This is done by using the "link" attribute inside the "head" tag of your HTML page. Inside of the link attribute, a .css file has to be added in order for it to link all of the styles inside onto the webpage. This saves a lot of code accross the webpages, saving lots of developing time. Its very efficient, and largely benefits bigger websites that use heavy amounts of styling accross webpages. Once you've edited and changed one .css file, you change all other webpages that are linked to it.

Useful information

What is CSS?

CSS (Cascading Style Sheets) is a markup language that is responsible for the looks of webpages. CSS allows you to change colours, fonts, layers and layouts, as well as adding animations and transitions to a website. These can be added to images, as well as buttons and loaders. Without CSS, webpages would look plain and boring.
